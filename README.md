Simple Signal Jammer 
A project using ESP32 and nRF24L01 to analyze and interfere with 2.4GHz wireless signals (WiFi & Bluetooth).

Features:
Dual Mode: Dedicated modes for WiFi (Channels 1-11) and Bluetooth (79 Hopping Channels).

Hardware Components:
Microcontroller: ESP32 wroom 32e 

Radio Module: nRF24L01 + PA + LNA.

Display: 0.96" OLED SSD1306 (I2C).

Peripherals: 2x Push Buttons, 1x Status LED, 10uF Capacitor (Recommended for power stability).
